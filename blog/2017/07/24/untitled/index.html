1 "__" started method

class A:
    def _single_method(self):
        print("In A single")

    def __double_method(self): # for mangling
        print("In A")

class B(A):
    def __double_method(self): # for mangling
        print("In B")

    def __double_method_2(self): # for mangling
        print("In B 2")

Some findings:

  1. '_doublemethod' in class A not exists in dir(A()), even 'A_doublemethod'.
  2. '_doublemethod' in class B exists in dir(B()) as 'B_doublemethod'
  3. '_doublemethod2' in class B not exists in dir(B())

So it seems like a '__' started method will only acceiable if and only if: its parent class has a same method defined.
